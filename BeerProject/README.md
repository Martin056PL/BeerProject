### General info
This is project to manage and support participants during beer day fest! 

### Technologies
Current technical features:
1. Getting and managing breweries and beers
2. Getting and managing users existing within application and their privileges
3. User authentication via login endpoint
4. Implemented gathering logs mechanism 
5. Implemented version control system for DB 
6. Added dockerfile and docker-compose to run app.
7. Added code validator to measure code coverage and detects codesmells

The list of available endpoints is generated by swagger () 

List of used technologies and tools:
* Java 11
* Spring Boot
* Spring data JPA
* Spring AOP
* Spring Security
* Lombok
* Docker
* Liquibase
* Swagger
* SLF4J
* H2
* MySQL
* Mockito
* JUnit 5
* SonarQube
